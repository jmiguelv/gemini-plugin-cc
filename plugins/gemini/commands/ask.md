# /gemini:ask

Use this command to ask the Gemini CLI a general question or request a task.

## Instructions
When this command is invoked, execute the `gemini` command in the shell with the user's query as an argument.

Example:
If the user runs `/gemini:ask how do I optimize this function?`, you should execute:
`gemini "how do I optimize this function?"`
