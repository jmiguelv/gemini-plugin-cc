# /gemini:review

Use this command to perform a code review using the Gemini CLI.

## Instructions
When this command is invoked, execute the following shell command to request a code review from the Gemini CLI, using `--approval-mode auto_edit` to allow it to apply improvements if requested:

`gemini --approval-mode auto_edit "Review the current uncommitted changes and provide feedback."`
